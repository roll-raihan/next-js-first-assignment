## Next.js Item App

A simple Next.js app with authentication, item listing, and protected routes.

### Features
- Landing page with 7 sections
- Mock authentication
- Public item listing
- Item details page
- Protected add item page
- Express.js backend

### Routes
- `/` Landing page
- `/login`
- `/items`
- `/items/[id]`
- `/items/add`

### Tech Stack
- Next.js App Router
- Express.js
- Tailwind CSS
